    heroku 发布项目流程以及常用命令

    1. heroku login //登录
    2. git clone project
    3. heroku create //创建heroku项目
    4. git push heroku master //将代码提交到heroku服务器上

    5. 声明一个Procfile 文件用来描述被执行的命令

    6. heroku ps:scale web=1 //确保有一个项目实例在运行

    7. heroku open //打开项目
    8. heroku logs –tail //查看项目运行状态
